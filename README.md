# JavaScript API Project

This project demonstrates the use of JavaScript to interact with various APIs, including fetching data for cat facts, jokes, dog images, and university search results. It uses **Axios** for making HTTP requests and **Bootstrap** for styling.

## Features

### 1. Cat Facts
- Fetches a random cat fact from the [Cat Fact API](https://catfact.ninja/fact).
- Displays the fact on the page when the "Get Cat Fact" button is clicked.

### 2. Jokes
- Fetches a random dad joke from the [icanhazdadjoke API](https://icanhazdadjoke.com/).
- Displays the joke on the page when the "Get Joke" button is clicked.

### 3. Dog Images
- Fetches a random dog image from the [Dog CEO API](https://dog.ceo/dog-api/).
- Displays the image on the page when the "Get Dog Image" button is clicked.

### 4. University Search
- Searches for universities based on a user-provided query using the [Hipolabs University API](https://universities.hipolabs.com/).
- Displays a list of matching universities.
- Includes a "Clear" button to reset the search input and results.

## Technologies Used
- **HTML**: For structuring the web page.
- **CSS**: For styling the page (via Bootstrap).
- **JavaScript**: For adding interactivity and making API calls.
- **Axios**: For making HTTP requests.
- **Bootstrap**: For responsive design and UI components.
